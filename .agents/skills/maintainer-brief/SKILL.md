# Maintainer Brief

Use this skill when preparing context for an issue response, pull request review, release decision, or follow-up investigation.

## Output Shape

- Summary
- Sources reviewed
- Evidence found
- Evidence gaps
- Maintainer decision points
- Follow-up tasks

## Rules

- Keep source links visible.
- Separate verified evidence from inference.
- Do not smooth over missing evidence.
- Prefer Markdown artifacts that can be reviewed outside chat.
